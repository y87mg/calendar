## Календарь на JS

Логику и стили календаря можно улучшать бесконечно

### Как подключить?

Всё просто. В файле calendar.js вызываем функцию createCalendar, которая принимает один параметр. В этот параметр нужно передать селектор, в конец которого будет добавлен календарь.

Пример:

`createCalendar('.main');`
