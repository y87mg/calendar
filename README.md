## Календарь на JS

  Логику и стили календаря можно улучшать бесконечно

### Как подключить?

  Всё просто. В файле calendar.js, вызываем функцию createCalendar и присваиваем её единственному параметру тег, в конец которого будет добавлен календарь.

  Пример:

  `const main = document.querySelector('.main');`

  `createCalendar(main);`
